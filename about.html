Welcome to student finance tracker

Purpose

The Student Finance Tracker is a simple and helpful web application created to help students take control of their spending. It allows users to record, organize, and review their expenses, making it easier to manage budgets and develop better money habits.

Key Features

  • Track expenses with detailed categorization
  • Multi-currency support with manual conversion rates
  • Advanced regex-based validation and search
  • Visual statistics and spending trends
  • Budget cap tracking with live notifications
  • Import/Export functionality for data backup
  • Fully keyboard accessible interface
  • Responsive design for mobile and desktop
  • Local data persistence using localStorage
